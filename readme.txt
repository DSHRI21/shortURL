# URL Shortner
It is an application for shortening the long URL or we can also use it as an encrypted URL.

## Setup
Download the code.
Create database with the name "urlshortner" import the database from "database" folder in downloaded file.
Update $config['base_url'] in 'application/config/config.php' with the path where you have downloaded it.

## Usage
1. For shortening the length of URL
2. For hiding the original URL
